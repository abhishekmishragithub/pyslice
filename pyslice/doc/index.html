3 Installation

Requires:

  • Python 1.5.2 with Distutils 0.9 or higher, or Python 2.x
  • Pdx required to regenerate documentation

This is a standard distutils install: 

  cp xyz-0.1.tar.gz myworkdir
  cd myworkdir
  gzip -cd xyz-0.1.tar.gz | tar xvf -
  cd xyz-0.1
  python setup.py build
  python setup.py install  #possibly as root
